# Guan-Ming's Claude Code Skills

Custom skills for Claude Code.

## Install

```bash
# Install skills
npx openskills install guan404ming/claude-code-skills --global -y

# Install global CLAUDE.md
curl -o ~/.claude/CLAUDE.md https://raw.githubusercontent.com/guan404ming/claude-code-skills/main/CLAUDE.md

# Remove all skills (optional)
rm -rf ~/.claude/skills/*
```
